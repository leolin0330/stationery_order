
#  自動化文具訂購

本專案是一個使用 **Python + Selenium** 開發的自動化文具訂購工具。

透過讀取 Excel 訂單資料，自動登入訂購平台，搜尋商品並加入購物車，
可大幅減少人工逐筆下單的時間並提升效率。

---

#  專案功能

本系統提供以下自動化功能：

- 自動登入訂購網站
- 從 Excel 批次讀取訂單資料
- 自動搜尋商品
- 自動加入購物車
- 記錄失敗訂單並輸出 Excel
- 完整自動化下單流程

---


#  系統流程

```
Excel 訂單資料
      │
      ▼
讀取 Excel (Pandas)
      │
      ▼
登入 ERP 系統
      │
      ▼
搜尋商品
      │
      ▼
加入購物車
      │
      ▼
輸出失敗訂單
```
